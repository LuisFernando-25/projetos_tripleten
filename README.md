# 🎧 Análise de Dados - Preferências Musicais

Este projeto foi desenvolvido como parte dos meus estudos em Análise de Dados.

## 📌 Objetivo

Analisar o comportamento dos usuários de um serviço de streaming de música e verificar se a atividade varia de acordo com:

- Cidade (Springfield e Shelbyville)
- Dia da semana (Monday, Wednesday, Friday)

## 🧠 Hipótese

A atividade dos usuários é diferente dependendo do dia da semana e da cidade.

---

## 🛠️ Tecnologias utilizadas

- Python
- Pandas

---

## 🔧 Etapas do projeto

### 1. Leitura dos dados
- Importação do dataset com Pandas

### 2. Pré-processamento
- Padronização dos nomes das colunas
- Tratamento de valores ausentes
- Remoção de duplicados
- Correção de inconsistências nos gêneros musicais

### 3. Análise
- Criação de função para calcular número de músicas tocadas
- Comparação entre cidades e dias da semana

---

## 📊 Resultados

Os dados mostram que:

- A atividade dos usuários varia ao longo da semana
- O comportamento é diferente entre as cidades
- Cada cidade apresenta picos de uso em dias distintos

---

## ✅ Conclusão

A hipótese foi confirmada.

A atividade dos usuários depende tanto da cidade quanto do dia da semana.
